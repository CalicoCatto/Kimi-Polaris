### 一些碎碎念
非常欣喜于 Kimi Palaris 的诞生。

其中有一些功能，例如时间线、文件夹，是 Gemini Voyager 给了我灵感。

也有一些其他的功能，源自于我自己在使用 Kimi 过程所受的烦扰。

在许多交互、图标的设计上也下了一些心思。但是还有很多没来得及优化的地方，会尽快修改的！

月光与音乐的元素，也带有很多隐喻，构成了该项目Dee主要风格，希望能让大家喜欢。
