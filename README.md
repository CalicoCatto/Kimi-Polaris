### 一些碎碎念
Kimi Palaris的诞生，源于我自己在使用Kimi模型时受到的一些界面操作烦扰以及及Gemini Voyager的启发。
其中有一些很好的功能，例如时间线、文件夹，是Gemini Voyager给了我灵感。
也有一些其他的功能，是源自于我自己在使用过程中的发现。
我是一个追求美感的INFJ，因此在许多交互、操作图标上也下了一些心思。
我很喜欢 Moonshot 与 Kimi 的诗意，我也用了很多月光与音乐的元素。希望能让大家喜欢。
